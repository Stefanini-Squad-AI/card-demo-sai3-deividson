🔐 Seguridad y Autenticación

Guía de Desarrollo de User Stories - CardDemo Legacy System

📋 Resumen del Módulo

ID: security-auth

Propósito: Gestionar autenticación y control de acceso al sistema CardDemo

Programas COBOL: COSGN00C, COUSR00C-03C

Archivo Principal: USRSEC (VSAM KSDS, 80 bytes)

🎯 Templates de User Stories

Como usuario del sistema, quiero iniciar sesión con User ID y Password para acceder al sistema CardDemo.
Como administrador, quiero crear nuevos usuarios con tipos ADMIN/USER/GUEST para controlar accesos.
